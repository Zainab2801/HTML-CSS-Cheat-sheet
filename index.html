HTML Table Reference

Basic Document Structure

Tag Name Description
<!DOCTYPE html> Document Type Declaration Defines the document type and version of HTML.
<html> HTML Element The root element of an HTML page.
<head> Head Element Contains meta-information about the HTML document. (not displayed to users)
<title> Title Element Defines the title of the HTML document (displayed in the browser's title bar or tab).
<meta> Meta Element Defines metadata about the HTML document (e.g., character set, author, description).
<link> Link Element Defines the relationship between the current document and an external resource (most used to link stylesheets css).
<style> Style Element Defines styles for the HTML document (usually contained within the <head> element).
<script> Script Element Defines a client-side script (e.g., JavaScript) for the HTML document.
<body> Body Element Contains the content of the HTML document (displayed to users).

Text Content

Tag Name Description
<h1> - <h6> Headings Defines HTML headings, with <h1> being the highest level and <h6> being the lowest.
<p> Paragraph Defines a paragraph of text.
<br> Line Break Inserts a line break in the text.
<hr> Horizontal Rule Defines a thematic break in the content (usually a horizontal line).
<strong> / <b> Strong Emphasis Defines important text (usually displayed in bold).
<i>/ <em> Italic Text Defines text that is emphasized (usually displayed in italics).
<sup> Superscript Defines text that is displayed as superscript (elevated above the baseline).
<sub> Subscript Defines text that is displayed as subscript (lowered below the baseline).

Links & Media

Tag Name Description
<a> Anchor Defines a hyperlink.
<img> Image Embeds an image in the document.
<figure> Figure Defines a self-contained content, like an image with a caption.
<audio> Audio Embeds an audio file in the document.
<video> Video Embeds a video file in the document.
<source> Source Specifies multiple media resources for video and audio elements.

Lists

Tag Name Description
<ul> Unordered List Defines a list of items in no particular order.
<ol> Ordered List Defines a list of items in a specific order.
<li> List Item Defines an item in a list.

Tables

Tag Name Description
<table> Table Defines a table.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines a cell in a table.
<th> Table Header Defines a header cell in a table.
<caption> Table Caption Defines a caption for a table.
<tfoot> Table Footer Defines a footer for a table.
<thead> Table Head Defines a header for a table.

Forms

Tag Name Description
<form> Form Defines an HTML form for user input.
<input> Input Defines an input control. Types: text, password, checkbox, radio, file, etc.
<textarea> Textarea Defines a multi-line text input control.
<button> Button Defines a clickable button.
<select> Select Defines a drop-down list.
<option> Option Defines an option in a drop-down list.
<label> Label Defines a label for an input element.

Semantic

Tag Name Description
<header> Header Defines a header for a document or section.
<nav> Navigation Defines a set of navigation links.
<main> Main Content Defines the main content of a document.
<article> Article Defines an independent piece of content.
<section> Section Defines a section in a document.
<aside> Aside Defines content aside from the main content.
<footer> Footer Defines a footer for a document or section.
<div> Division Defines a division or section in a document.
<span> Span Defines a section in a document. Inline Container

CSS Cheatsheet

Selectors

Selector Name Description
* Universal Selector Selects all elements.
element Type Selector Selects all
elements.
element.class Class Selector Selects all elements with the specified class.
element#id ID Selector Selects a single element with the specified ID.
element::before Pseudo-element Selector Selects a specific part of an element.
element::after Pseudo-element Selector Selects a specific part of an element.
element:hover Pseudo-class Selector Selects an element when it is hovered over.

Box Model

Property Name Description
width Width Defines the width of the content area.
height Height Defines the height of the content area.
padding Padding Defines the space between the content and the border. Inner spacing
border Border Defines the border around the element.
margin Margin Defines the space outside the border. Outer spacing

Display & Positioning

Property Name Description
display Display Defines the display behavior of an element. Block, inline, flex, etc.
position Position Defines the positioning method for an element. Static, relative, absolute, fixed, sticky.
top Top Defines the top position of a positioned element. Moves element down from the top.
right Right Defines the right position of a positioned element. Moves element left from the right.
bottom Bottom Defines the bottom position of a positioned element. Moves element up from the bottom.
left Left Defines the left position of a positioned element. Moves element right from the left.
z-index Z-Index Defines the stack order of a positioned element.
overflow Overflow Defines what happens when content overflows an element's box. (Hidden, scroll, auto)

Flexbox

Property Name Description
display Display Defines the display behavior of an element as a flex container.
flex-direction Flex Direction Defines the direction of the flex items (row, column).
justify-content Justify Content Defines the alignment of flex items along the main axis.
align-items Align Items Defines the alignment of flex items along the cross axis.
flex-wrap Flex Wrap Defines whether flex items should wrap onto multiple lines.

Typography

Property Name Description
font-family Font Family Defines the font family for text.
font-size Font Size Defines the size of the font.
font-weight Font Weight Defines the weight (boldness) of the font.
line-height Line Height Defines the height of a line box.
text-align Text Align Defines the horizontal alignment of text. left, right, center, justify
text-decoration Text Decoration Defines the decoration added to text (underline, overline, line-through).
text-transform Text Transform Defines the capitalization of text (uppercase, lowercase, capitalize).
letter-spacing Letter Spacing Defines the spacing between characters in text.

Colors & Backgrounds

Property Name Description
color Color Defines the text color of an element.
background-color Background Color Defines the background color of an element.
background-image Background Image Defines the background image of an element.
background-size Background Size Defines the size of the background image. Cover, contain, or specific dimensions.
opacity Opacity Defines the transparency level of an element.

Miscellaneous

Property Name Description
box-shadow Box Shadow Defines the shadow effect around an element's box.
border-radius Border Radius Defines the radius of an element's corners.
transition Transition Defines the transition effect when a property changes.
transform Transform Defines the transformation (rotation, scaling, etc.) of an element.
border-collapse Border Collapse Defines whether table borders are collapsed into a single border (looks cleaner) or separated.
